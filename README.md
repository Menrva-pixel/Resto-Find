## Run Locally

Clone the project

```bash
  git clone https://github.com/Menrva-pixel/Resto-Find
```

Go to the project directory

```bash
  cd Project Directory (RestoFind / Resto-Find)
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
development mode :
  npm run start-dev
```

```bash
create local server :
  npm run serve
```
End-to-End Testing

```bash
create local server :
node server.js

test command :
npm run e2e
```
